Introduction

Neura is a statically-typed, Python-based programming language with a clean and expressive syntax. It draws inspiration from C-style structure, modern scripting languages, and Pythonic readability. Designed to be beginner-friendly yet powerful, Neura supports strong type annotations, readable control flow, and simplified syntax for rapid development. 

fn greet(name: str) {
    say("Hello, ", name);
}

greet("Neura");

x: int = 10;
y: float = 20.5;
name: str = "Alice";
is_valid: bool = true;

if x > 5 {
    say("x is greater than 5");
} else {
    say("x is 5 or less");
}

for i in 0..5 {
    say("Loop value:", i);
}

This example demonstrates Neura's basic syntax including function declarations, conditionals, loops, and type-safe variables.

Variables

In Neura, variables are declared with the syntax variable_name: type = value.

Variable Declaration

Here's an example of declaring different types of variables:

x: int = 10;            // Integer variable
y: float = 100.12;      // Float variable
name: str = "John";    // String variable
is_ready: bool = true;    // Boolean variable
anything: dynamic = 10;   // Dynamic variable

Reassigning Dynamic Variables

You can reassign values to dynamic variables:

anything = "Now I'm a string!";  // Reassigning a dynamic variable

Collections

Neura also supports collections like lists and hashes (similar to maps in other languages):

fruits: list = ["apple", "mango", "orange"];   // List
nested: hash = {"person": {"name": "Mark", "age": 20}, "scores": [95, 85, 75]};  // Hash (HashMap)

Data Types

Neura supports the following data types:

int
Integer numbers for whole-value arithmetic operations and counting.
float
Floating-point numbers for precise decimal calculations and scientific computations.
str
String data enclosed in double quotes with built-in manipulation methods.
bool
Logical values represented as true and false for conditional operations.
list
Ordered collections of elements that can be accessed by index and modified dynamically.
hash
Key-value mappings for efficient lookups and structured data organization.
dynamic
Flexible type that can hold any value and change types during runtime for adaptive programming.

Control Flow

Neura provides robust control flow structures including conditional statements and various loop types to control program execution.

Conditional Statements

Use if, else if, and else statements with curly braces to control program flow based on conditions:

if user_age > 18 {
    say("User is an adult.");
} else if user_age == 18 {
    say("User is exactly 18 years old.");
} else {
    say("User is a minor.");
}

While Loop

Execute a block of code repeatedly as long as a condition is true:

count: int = 10;
while count > 0 {
    say(count);
    count = count - 1;
}

For Loop

Iterate through a range of values with optional step increment using the by keyword:

// for loop with range and step
for i in 0..10 by 2 {
    say("Iteration:", i);
}

Foreach Loop

Iterate through all elements in a collection such as an array or list:

// foreach loop
fruits = ["apple", "banana", "cherry"];
foreach fruit in fruits {
    say("Fruit:", fruit);
}
Control Flow Tips
  • All control structures in Neura use curly braces {} to define their scope
  • Range notation uses two dots .. to specify start and end values
  • The by keyword allows custom increments in for loops

Functions

Functions are defined using the fn keyword. Here is an example:

fn greet(name: str) {
    say("Hello, ", name);
}

You can call the function like this:

greet("Alice");

Built-in Functions

Neura provides several built-in functions, grouped by their purpose:

Input Functions
  • ask(prompt): Prompts the user for input and returns the value.
Output Functions
  • say(message): Prints values to the console.
  • notify(message): Sends a short notification to the user.
Type Conversion
  • asString(): Converts a variable to a string.
  • asInt(): Converts a variable to an integer.
  • asFloat(): Converts a variable to a float.
  • asBool(): Converts a variable to a boolean.
Utility Functions
  • type(value): Returns the type of a variable as a string.
  • wait(seconds): Pauses execution for a specified number of seconds.
  • length(value): Returns the number of elements in a string, list, or hash.
String Functions
  • trim(text): Removes whitespace from both ends of the string.
  • upperCase(text): Converts all characters in the string to uppercase.
  • lowerCase(text): Converts all characters in the string to lowercase.

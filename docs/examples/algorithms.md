# Algorithm Examples

This section contains a collection of algorithm implementations in Echo, demonstrating how to solve various coding challenges. All source code is available in the [repository's examples folder](https://github.com/deekshith-poojary98/echo/tree/main/examples).

## Array & List Manipulation

Learn how to work with arrays and lists through practical algorithm implementations.

- [Add Two Numbers](https://github.com/deekshith-poojary98/echo/blob/main/examples/add_two_numbers.echo) - Add two numbers represented as linked lists
- [Container With Most Water](https://github.com/deekshith-poojary98/echo/blob/main/examples/container_with_most_water.echo) - Find two lines that form a container with maximum area
- [Remove Duplicates from Sorted Array](https://github.com/deekshith-poojary98/echo/blob/main/examples/remove_duplicates_sorted_array.echo) - Remove duplicates in-place from sorted array
- [Remove Element](https://github.com/deekshith-poojary98/echo/blob/main/examples/remove_element.echo) - Remove all instances of a value in-place
- [Merge Two Sorted Lists](https://github.com/deekshith-poojary98/echo/blob/main/examples/merge_two_sorted_lists.echo) - Merge two sorted linked lists
- [Merge K Sorted Lists](https://github.com/deekshith-poojary98/echo/blob/main/examples/merge_k_sorted_lists.echo) - Merge k sorted lists into one sorted list
- [Jump Game II](https://github.com/deekshith-poojary98/echo/blob/main/examples/jump_game_ii.echo) - Find minimum jumps to reach last index
- [Remove Nth Node From End of List](https://github.com/deekshith-poojary98/echo/blob/main/examples/remove_nth_from_end.echo) - Remove the nth node from end of list

## Getting Started with Examples

To run any of these examples locally:

```bash
python echolang.bat path/to/example.echo
```

Or clone the repository and run directly:

```bash
git clone https://github.com/deekshith-poojary98/echo.git
cd echo
echo examples/two_sum.echo
```

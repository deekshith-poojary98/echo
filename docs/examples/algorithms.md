# Algorithm Examples

This section contains a collection of algorithm implementations in Echo, demonstrating how to solve various coding challenges. All source code is available in the [repository's examples folder](../../../examples/).

## Array & List Manipulation

Learn how to work with arrays and lists through practical algorithm implementations.

- [Add Two Numbers](../../../examples/add_two_numbers.echo) - Add two numbers represented as linked lists
- [Container With Most Water](../../../examples/container_with_most_water.echo) - Find two lines that form a container with maximum area
- [Remove Duplicates from Sorted Array](../../../examples/remove_duplicates_sorted_array.echo) - Remove duplicates in-place from sorted array
- [Remove Element](../../../examples/remove_element.echo) - Remove all instances of a value in-place
- [Merge Two Sorted Lists](../../../examples/merge_two_sorted_lists.echo) - Merge two sorted linked lists
- [Merge K Sorted Lists](../../../examples/merge_k_sorted_lists.echo) - Merge k sorted lists into one sorted list
- [Jump Game II](../../../examples/jump_game_ii.echo) - Find minimum jumps to reach last index
- [Remove Nth Node From End of List](../../../examples/remove_nth_from_end.echo) - Remove the nth node from end of list

- [First Missing Positive](../../../examples/first_missing_positive.echo) - Find first missing positive integer
- [Find First Occurrence](../../../examples/find_first_occurrence.echo) - Find first occurrence of substring

- [Next Permutation](../../../examples/next_permutation.echo) - Generate next lexicographic permutation

## Getting Started with Examples

To run any of these examples locally:

```bash
python echolang.bat path/to/example.echo
```

Or using the Echo CLI directly:

```bash
echo path/to/example.echo
```

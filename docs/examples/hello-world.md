# Hello World

## Overview
A minimal Echo program.

## Syntax
```echo
say("Hello, Echo!");
```

## Example
```echo
say("Hello, Echo!");
```

## Output
```text
Hello, Echo!
```

## Notes
- `say(...)` is the standard output helper.
- Even a one-line program still needs a semicolon.

## Common Mistakes
- Forgetting the semicolon
